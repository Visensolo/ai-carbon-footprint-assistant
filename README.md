AI 碳足迹计算 + 行为优化助手
一个轻量化网页工具，帮普通人快速计算日常碳足迹，并获得个性化、可落地的低碳优化建议，呼应联合国「气候行动」议题，助力全民低碳意识提升。
🌟 项目背景
全球气候变化日益严峻，低碳生活成为应对气候危机的重要举措，但普通人普遍存在碳排放认知空白：不清楚日常出行、饮食行为的碳排放量差异，也没有明确的低碳优化方向。本项目旨在解决这一痛点，提供一个零门槛、易操作的碳足迹计算工具，让普通人轻松参与低碳行动。
✨ 核心功能
场景化碳足迹计算
内置高频场景一键选择（出行 / 饮食），支持数值灵活调整（如可自定义出行里程、食材用量）
预设权威碳排放系数，快速完成精准碳排放量估算
AI 个性化优化建议
针对用户行为生成贴合日常习惯的低碳建议，避免空泛说教
标注可减少的碳排放量，让优化效果直观可见
极简操作体验
纯网页端工具，无需安装任何软件，打开即可使用
界面简洁清晰，一键完成「选择场景 - 计算碳排 - 查看建议」全流程
🚀 使用说明
打开项目根目录下的 index.html 文件（直接用浏览器打开即可）
在「出行场景」或「饮食场景」中，点击选择常用场景，并根据自身情况调整数值
点击「计算碳足迹」按钮，查看碳排放量结果及分类统计
查看下方由 AI 生成的个性化低碳优化建议，可直接参考执行
