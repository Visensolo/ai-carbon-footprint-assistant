# AI 碳足迹计算 + 行为优化助手

一个轻量化网页工具，帮普通人快速计算日常碳足迹，并获得个性化、可落地的低碳优化建议，呼应联合国「气候行动」议题，助力全民低碳意识提升。

---

## 🌟 项目背景
全球气候变化日益严峻，低碳生活成为应对气候危机的重要举措，但普通人普遍存在碳排放认知空白：不清楚日常出行、饮食行为的碳排放量差异，也没有明确的低碳优化方向。本项目旨在解决这一痛点，提供一个零门槛、易操作的碳足迹计算工具，让普通人轻松参与低碳行动。

## ✨ 核心功能
**场景化碳足迹计算**
- 内置高频场景一键选择（出行/饮食），支持数值灵活调整（如可自定义出行里程、食材用量）
- 预设权威碳排放系数，快速完成精准碳排放量估算

**AI 个性化优化建议**
- 针对用户行为生成贴合日常习惯的低碳建议，避免空泛说教
- 标注可减少的碳排放量，让优化效果直观可见

**极简操作体验**
- 纯网页端工具，无需安装任何软件，打开即可使用
- 界面简洁清晰，一键完成「选择场景-计算碳排-查看建议」全流程

## 🚀 使用说明
- 打开项目根目录下的 `index.html` 文件（直接用浏览器打开即可）
- 在「出行场景」或「饮食场景」中，点击选择常用场景，并根据自身情况调整数值
- 点击「计算碳足迹」按钮，查看碳排放量结果及分类统计
- 查看下方由 AI 生成的个性化低碳优化建议，可直接参考执行

## 📊 项目后期发展规划

### 🤖 AI 核心能力升级
未来将进一步升级 AI 驱动的碳足迹计算能力，实现多源数据自动解析，通过 OCR 与 NLP 技术对接出行、外卖、用电等生活数据，让碳足迹计算告别手动输入。同时结合地理位置、天气、交通等实时信息，动态优化碳排放系数，让计算结果更加精准贴近真实生活场景。

基于用户习惯、身份特征与减排目标，AI 将自动生成可落地、可执行的个性化低碳方案，包括日常低碳建议、长期减排计划等。系统还将支持自然语言对话交互，用户可通过文字或语音直接描述行为，由 AI 自动解析、计算并给出优化建议，打造真正智能的低碳生活助手。

### 🌐 场景拓展与生态联动
项目将持续拓展碳足迹计算场景，覆盖居家、消费、交通、服饰、快递等更多生活领域，通过 AI 场景识别自动匹配碳排放数据，构建完整的个人碳足迹画像。同时接入电商、生活服务平台，为用户提供低碳商品推荐与绿色消费指引。

未来还将搭建社区化低碳激励体系，通过减排排行、低碳挑战、AI 生成勋章证书等方式提升用户参与感，并将个人减排量与公益植树、碳汇项目联动，让低碳行为真正产生社会价值，形成可持续的绿色生态闭环。
