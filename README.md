# AI 碳足迹计算 + 行为优化助手

一个轻量化网页工具，帮普通人快速计算日常碳足迹，并获得个性化、可落地的低碳优化建议，呼应联合国「气候行动」议题，助力全民低碳意识提升。

---

## 🌟 项目背景
全球气候变化日益严峻，低碳生活成为应对气候危机的重要举措，但普通人普遍存在碳排放认知空白：不清楚日常出行、饮食行为的碳排放量差异，也没有明确的低碳优化方向。本项目旨在解决这一痛点，提供一个零门槛、易操作的碳足迹计算工具，让普通人轻松参与低碳行动。

## ✨ 核心功能
1.  **场景化碳足迹计算**
    - 内置高频场景一键选择（出行/饮食），支持数值灵活调整（如可自定义出行里程、食材用量）
    - 预设权威碳排放系数，快速完成精准碳排放量估算
2.  **AI 个性化优化建议**
    - 针对用户行为生成贴合日常习惯的低碳建议，避免空泛说教
    - 标注可减少的碳排放量，让优化效果直观可见
3.  **极简操作体验**
    - 纯网页端工具，无需安装任何软件，打开即可使用
    - 界面简洁清晰，一键完成「选择场景-计算碳排-查看建议」全流程

## 🚀 使用说明
1.  打开项目根目录下的 `index.html` 文件（直接用浏览器打开即可）
2.  在「出行场景」或「饮食场景」中，点击选择常用场景，并根据自身情况调整数值
3.  点击「计算碳足迹」按钮，查看碳排放量结果及分类统计
4.  查看下方由 AI 生成的个性化低碳优化建议，可直接参考执行

# 🚀 项目后期发展规划


## 1. 🤖 AI 核心能力升级

### 1.1 碳足迹数据自动采集与精准计算
- **多源数据自动解析**：对接用户授权的出行、外卖、用电账单等数据，通过OCR识别与NLP解析技术，自动提取行为数据并完成碳足迹计算，告别手动输入。
- **动态系数适配**：结合用户地理位置、天气、交通数据，动态调整碳排放系数（如不同地区电网碳强度、不同路况下的车辆油耗），大幅提升计算精准度。

### 1.2 个性化低碳方案生成
- **场景化定制计划**：基于用户身份（学生/职场/独居）、行为习惯与减排目标，由AI生成可执行的「周度低碳日历」，标注每次行为的减排量与替代方案。
- **对话式交互体验**：接入大语言模型，支持自然语言对话输入（如“我今天开车上班10公里，点了一份外卖”），AI自动解析、计算并给出建议。

### 1.3 智能低碳顾问功能
- 支持用户提问“如何在不影响生活质量的情况下减排？”“不同出行方式的长期碳影响对比”，AI提供专业、易懂的解答与对比分析。

---

## 2. 🌐 场景拓展与生态联动

### 2.1 多生活场景碳足迹覆盖
- 拓展居家用电/用水、快递包装、旅行住宿、服装消费等高频场景，通过AI场景识别模型，自动匹配碳排放系数，构建完整的用户碳足迹画像。
- 接入电商公开数据，为用户购物订单生成“碳标签”，推荐低排放替代商品。

### 2.2 社区化低碳激励体系
- 搭建用户减排排行榜、低碳挑战活动，AI根据用户减排量生成专属勋章/证书，增强参与感。
- 对接公益项目，将用户减排量量化为碳汇项目捐赠，实现“减排即公益”的闭环。

---

## 3. 📊 数据可视化与趋势分析

### 3.1 AI驱动的碳足迹报告
- 定期生成月度/年度碳足迹报告，通过AI分析用户减排趋势与高碳行为集中点，以图表+文字解读的形式呈现，清晰展示用户低碳进展。
- 对比同场景用户平均碳排放水平，AI给出针对性优化建议（如“你的通勤碳排放量高于同校用户平均水平，推荐尝试拼车或骑行”）。

### 3.2 隐私安全保障
- 采用AI联邦学习技术，在不泄露用户原始数据的前提下，实现碳足迹数据的统计分析与模型优化，保障用户隐私安全。
